# Users project

## Clone repository
```
git clone https://github.com/jumatosk/users.git
```

## Access Project Folder
```
cd users
```

## Open Project
```
code .
```

## Install dependencies
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
